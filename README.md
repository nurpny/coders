# Coders
##### A social networking website for programmers.
Coders is a social networking website that allows programmers to connect with other programmers to work on pair programming/

## Technologies Used
Front-end: Typescript, React, Redux, React-Redux
Back-end: MongoDB/Mongoose, Node.js, Express

## Getting Started
1.  `git clone` this repository to your local computer.
2.  In your computer's terminal, run `npm install`.
3.  Run `npm run start-dev`.

### 
Sign up and click on languages that you want to collaborate with others on.
